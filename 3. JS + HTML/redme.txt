
Разработать приложение: Редактор заметок с тегами.

Действия на странице:

Заметки можно создать 
Заметки можно отредактировать нажав на соответствующую заметку в списке
Заметки можно удалить нажав на соответствующую кнопку

На странице должен быть фильтр по тегу выполненный в виде строки поиска
Как только пользователь поместил курсор в поле для ввода тега под ним должен появиться список в котором находятся все теги которые есть в приложении
При вводе в поля фильтра список тегов также фильтруется
При нажатии на Тег он помещается в фильтр 
Как только в списке остался один Тег список скрывается 
Или пользователь покинул поле ввода текста


Во время создания и редактирования заметки пользователь может создавать новые теги, используя символ #.
Например: пользователь вводит текст “I go to #shop”. По мере (окончании) ввода должен создаться соответствующий тег и отобразиться в списке под текстовым полем.
При редактировании заметки все слова, являющиеся тегами, должны подсвечиваться.

При удалении тега из заметки он удаляется из приложения

Плюсом будет красивое оформление
